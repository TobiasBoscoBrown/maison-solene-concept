# Maison Solène — boutique hotel website concept

A responsive, single-page website concept for a fictional boutique hotel in
Charlevoix, Québec. Built as a free sample by **BuildMyTribe** for the Studio Circa
"Website Designer (UI) for Boutique Hotel Project" opportunity on Contra.

- Hand-built HTML, CSS, and vanilla JS (no framework, no build step)
- Editorial layout: storytelling-led hero, rooms, dining, Nordic baths, experiences, gallery, booking
- Scroll reveal animations (respects `prefers-reduced-motion`)
- Accessible: skip link, semantic landmarks, alt text, visible focus, keyboard-friendly nav
- Self-hosted imagery, fully responsive (desktop + mobile)

This is a design concept for a fictional property and is not affiliated with any real hotel.
